# DBMS 성능 측정 방법

## 1. 고가용성을 위한, DBMS 성능 측정 방법 개요
:::tip[개념]
DBMS의 성능 측정을 위해 벤치마크 및 시나리오 기반의 테스트등을 수행하여 성능 특성을 파악하는 활동
:::
:::tip[ISO/IEC 25010]
품질특성을 평가를 위한 모델 ISO/IEC 25010의 실행 효율성 항목을 활용하여 "시간효율성", "자원활용성", "기억용량" 등을 측정
:::

ㅡ DBMS 성능 지표를 설정하여 도메인 분석 후 평가 수행
